# Daftar File Referensi

Folder ini berisi lima jurnal utama yang digunakan untuk laporan KKP Chatbot Antrik.

## Keterangan Highlight

File PDF yang sudah diberi highlight berada pada folder `referensi/highlighted/`. Highlight diberikan pada kalimat atau poin penting, bukan hanya pada kata kunci.

- Highlight kuning: kalimat atau poin penting umum dari jurnal.
- Highlight merah: kalimat atau poin yang dipakai langsung untuk memperkuat isi laporan.

## Jurnal Utama yang Digunakan

1. `caldarini-2022-literature-survey-chatbots.pdf`
   - Caldarini, Jaf, dan McGarry (2022)
   - DOI: https://doi.org/10.3390/info13010041
   - Peran: klasifikasi dan perkembangan chatbot, termasuk rule-based, retrieval, dan generative chatbot.

2. `chandrakala-2023-intent-recognition-pipeline.pdf`
   - Chandrakala, Bhardwaj, dan Pujari (2023)
   - DOI: https://doi.org/10.1007/s41870-023-01642-8
   - Peran: intent recognition dan entity extraction pada conversational AI.

3. `warto-2024-systematic-literature-review-ner.pdf`
   - Warto dkk. (2024)
   - DOI: https://doi.org/10.19139/soic-2310-5070-1631
   - Peran: Named Entity Recognition dan entity extraction.

4. `martinengo-2023-conversational-agents-health-care.pdf`
   - Martinengo dkk. (2023)
   - DOI: https://doi.org/10.2196/50767
   - Peran: conversational agent di layanan kesehatan, framework, etika, dan keamanan data.

5. `klug-2024-clinical-nlp-patient-journey.pdf`
   - Klug dkk. (2024)
   - DOI: https://doi.org/10.1186/s12911-024-02641-w
   - Peran: penerapan NLP klinis untuk mengubah teks kesehatan menjadi informasi terstruktur.

File `admin,+F-07.pdf` adalah contoh referensi dari dosen terkait pengolah bahasa alami dengan scanner, parser, aturan produksi, translator, evaluator, dan jawaban akhir.
